# 🤖 AI 智能系统

Priaxis 内置双 AI 助手系统，支持自然语言对话和智能任务管理。

---

## 双角色设计

### 🐱 喵友 (Friend AI)

温暖陪伴型助手，本地规则驱动。

**能力**:
- 每日问候，询问睡眠状态
- 根据回复设置今日精力值 (0-100)
- 轻量级任务提醒
- 情绪支持和鼓励

**特点**:
- 无需 API 密钥，完全离线
- 关键词检测 + 模板匹配
- 响应快速，不消耗配额

---

### 🧑‍🏫 教练 (Coach AI)

专业指导型助手，Gemini/OpenAI 驱动。

**能力**:
- 任务优先级分析 (MoSCoW)
- SMART 目标检测
- 复盘总结生成
- 深度任务拆解
- 周/月报生成

**需要**: 配置 bridge 侧 `GEMINI_API_KEY` 或 `OPENAI_API_KEY`（不要放在前端 `VITE_*`）

---

## Function Calling

AI 可以直接操作你的任务系统：

| 指令示例 | AI 调用 |
|---------|--------|
| "帮我加一个任务叫做读书" | `add_task` |
| "删除那个运动任务" | `delete_task` |
| "昨晚睡得很好，8小时" | `update_energy(90)` |
| "我现在状态怎么样？" | `get_player_status` |
| "列出我的所有任务" | `get_tasks` |

## 智能干预系统

后台监测引擎每 10 分钟自动检测：

| 触发条件 | 干预类型 |
|---------|---------|
| 空闲过久 | 提醒开始工作 |
| DDL 推迟多次 | 询问障碍 |
| 完成率过低 | 建议调整目标 |
| 精力值低 | 建议休息 |

---

## 策略引擎

### MoSCoW 优先级

- **M**ust have — 必须完成
- **S**hould have — 应该完成
- **C**ould have — 可以完成
- **W**on't have — 暂不考虑

### SMART 检测

自动检测任务是否符合：
- **S**pecific (具体)
- **M**easurable (可衡量)
- **A**chievable (可实现)
- **R**elevant (相关)
- **T**ime-bound (有时限)
